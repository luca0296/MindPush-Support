# MindPush — Support

Official support page for MindPush: Digital Discipline.

## App Description
MindPush is a discipline enforcement system that requires physical or mental output to earn screen time. Complete push-ups, squats, or deep work sessions to unlock your apps. No output, no access. Simple. Brutal. Effective.
